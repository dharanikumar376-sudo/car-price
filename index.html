📖 Project Abstract

This project presents a machine learning solution for predicting used car prices in the Indian market. Using a dataset of 1,428 records from CarDekho, six regression models were trained and evaluated. The XGBoost model achieved the best performance with R²=0.95 and RMSE=1.4 Lakhs. Key price factors include engine power, engine CC, and year of manufacture. The system is deployed as an interactive web application for real-time price estimation.
